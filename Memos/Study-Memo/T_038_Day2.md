**因子策略**

美股常用 smart beta：

| 大市值 | 市值最大                                |
| ------ | --------------------------------------- |
| 小市值 | 市值最小                                |
| 高价值 | 账面/市值 最高 (市净率倒数)             |
| 高动量 | 过去12个月总return最高                  |
| 低波动 | 相对大盘beta最低                        |
| 高质量 | 高利润、高成长、稳定优良的管理、正alpha |
